# Projet Algorithmique et Structures de Données - LMI 2

## Domaine Applicatif : (a) Système de gestion de scolarité

Ce projet implémente un système de gestion de scolarité en C. Il permet de manipuler et de comparer les performances de 5 structures de données différentes pour le stockage d'enregistrements d'étudiants.

**Binôme :** Mouhamadou Hady MBAYE & Astou SEYE
**Professeur :** Dr. Abdoulaye DIALLO

### Structures implémentées
1. Tableau statique
2. Tableau dynamique de pointeurs
3. Liste doublement chaînée
4. Arbre Binaire de Recherche (ABR)
5. Tas Binaire (File de priorité MAX)
